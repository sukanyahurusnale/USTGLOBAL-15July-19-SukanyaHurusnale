Let's talk a little about filters when working with pictures and images!

Filters are very useful for retouching and photo correction. Filters can also be used to create special effects or unique transformations of images. In addition to the filters provided by Adobe, you can use some third-party plug-ins. After installation, these filter plug-ins are displayed at the bottom of the Filter menu.

    You can apply filters in three ways:

  • Filter menu

    • Contains all available filters and allows you to apply them separately.

  • Filter Gallery

    • Displays the thumbnails of the sample filter application in the form of a panel. "Filter Gallery" consistently allows you to apply several filters and use separate filters more than once. It is possible to change the order of the filters and adjust each filter to achieve the desired effect. Because of its flexibility, the Filter Gallery is often the best method for applying filters. However, in the "Filter Gallery" dialog box, not all the filters available in the "Filter" menu are available.

  • Filters panel

    • Displays the thumbnails of the filter application samples from the Filters menu. In the Expert mode, you can apply filters to each photo individually using the Filters panel.

Tips for applying filters

The following tips will help you understand the technique of applying filters to images.

Preview the results of the action of the filters. Applying filters to a large image can take a long time. You can quickly see the sample application filter in the "Gallery of filters." Most filters allow you to preview the result in the "Filter Parameters" dialog box in the document window. After evaluating the possible result, you can apply the filter or abandon it without losing time.

Filters are applied only to the active part of the image. Filters only affect the active, visible layer or selected area of ​​the layer.

Filters do not apply to all images. Some filters can not be used for images in grayscale, bit mode, or indexed color mode. Many filters do not affect 16-bit images.

It is possible to repeat the previous filter. Filter. The last one applied is displayed at the top of the "Filters" list. This filter can be repeated again to increase the dramatic effect of the image with the available settings.

Tips for creating visual effects with filters

Use the following techniques to create special visual effects with filters.

Feathering the edges of the filter When applying the filter to the selection area, you can soften the edges of the filter by setting the feathering selection before applying the filter.

Apply multiple filters to create a combined effect. It is possible to apply filters as a sequence to individual layers, and to several layers simultaneously, combining the effects they create. When you select different blend modes in the Layers panel, the selected effects are combined. For the filter to have its effect, the layer must be visible and must contain pixels.

Create textures and backgrounds. Applying effects to solid colors or grayscale images allows you to create many background images and textures. After that, you can apply the blur effect to them. Despite the fact that some filters (for example, the filter "Glass") slightly affect the solid colors, other filters create interesting effects. For solid colors, you can use the filter "Add Noise", "Chalk and Coal", "Clouds", "Magic Pencil", "Overlapping Clouds", "Glass", "Graphic Pencil", "Halftone Pattern", "Mezzo Tinto "," Postal Paper "," Pointillism "," Reticulation "," Rough Pastel "," Sponge "or" Drawing on the back ". You can also use any filter from the "Texture" submenu.

Improving the quality and integrity of the image. Eliminate the shortcomings, modify and enhance the artistic expressiveness of images or create a group of similar images can be applied to them the same filters.
